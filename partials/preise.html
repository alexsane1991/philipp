„Benötigen Sie eine Datenrekonstruktion von beschädigten Videofiles?“

Video-Rekonstuktion und Preise

Vorbereitung:

Senden Sie mir unverbindlich eine ihrer korrupten Dateien sowie eine oder mehrere „gesunde“ Dateien per WeTransfer. Sofern der Inhalt rekonstruierbar ist bekommen sie eine Vorschaudatei per Mail zugesendet. Nun können sie entscheiden ob sie ein vollständige Datenrekonstruktion in Auftrag geben wollen. Zunächst können Sie mir ein paar betroffene Dateien gratis zur Prüfung übermitteln. Wie bereits erwähnt sollten Sie mir zusätzlich zur defekten Datei eine oder mehrere „gesunde“ Clips aus gleicher Quelle als Referenz übermitteln.

Zur Beschleunigung der Datenrekonstruktion und als Basis zur Angebotserstellung sind folgende Angaben hilfreich.

  • Wie viele Videodateien gingen verloren und wie groß sind die Dateien bzw. welche Dauer hatte ihr Inhalt.
  • Mit welchem Kameramodel oder welcher Software wurden die Daten generiert. In welchem Dateicontainerformat liegen die Dateien vor? (MOV, AVI, MP4 etc.)
  • Mit welchen Settings wurden diese Dateien generiert? (z.B. H.264, 1920 x 1080, 25fps, Audio: 48khz , 16bit Stereo)
  • Ab wann sollen die Videos verfügbar sein?
  • Falls die Daten konvertiert werden müssen. In welcher Norm werden diese gewünscht.
  • Benötigen Sie nur Bild, Ton oder beides.

Kosten: Datenrekonstruktion

Sollte die Rekonstruktion möglich sein, erhalten Sie von mir ein Angebot sowie das rekonstruierte File als Vorschau. Benötige Sie die Datenrekonstruktion von nur einer einzigen Datei belaufen sich die Kosten auf 100€ (Preis inkl. Mwst.). Handelt es sich um mehrere Dateien so verrechne ich 25€ für jede weitere Rekonstruktion wobei Mengenrabatt möglich ist.
